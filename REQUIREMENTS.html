Distill Local Model Onboarding

TL;DR: make Distill default to its own local model, start the local OpenAI-compatible server on demand, and keep external APIs available through explicit config.

Default
Provider defaults to local for fresh and existing config resolution.
Mac
Apple Silicon uses MLX model samuelfaj/distill-1.7B-4bit-MLX.
Other OS
Linux, Windows, and non-Apple-Silicon Mac use llama.cpp GGUF.
Concurrency
Default max concurrent local requests is 5, configurable.

Scope

In scope
